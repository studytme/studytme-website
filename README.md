## Installation


1. Clone the repo
   ```sh
   git clone https://github.com/studytme/studytme-website.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
    npm run dev
   ```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
